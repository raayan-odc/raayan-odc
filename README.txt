
RAAYAN ENTERPRISES - Website Package

Instructions:
1. Upload the contents to a GitHub repository (e.g., raayan.github.io)
2. Go to GitHub > Settings > Pages > Select main branch > root folder
3. Visit https://yourusername.github.io to see your website live
4. To connect domain:
   - Login to GoDaddy
   - Add a CNAME record pointing www to yourusername.github.io
   - Add custom domain in GitHub Pages settings

Thanks for using this package!
